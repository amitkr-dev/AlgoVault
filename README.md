# AlgoVault

A personal DSA Knowledge Base built to organize algorithm patterns, notes, coding templates, question references, and revision resources in one place.

## Overview

AlgoVault is a learning and revision platform designed for Data Structures and Algorithms preparation.

The goal is to keep all DSA patterns, notes, and important problem references organized in a single workspace instead of scattered across notebooks and documents.

## Features

### Topic Management

* Add Topics
* Rename Topics
* Delete Topics

Examples:

* Arrays
* Strings
* Trees
* Graphs
* Dynamic Programming

### Pattern Management

* Add Patterns
* Rename Patterns
* Delete Patterns

Examples:

* Hashing
* Two Pointers
* Sliding Window
* Prefix Sum
* Binary Search

### Notes Editor

Store:

* Theory
* Templates
* Revision Notes
* Code Snippets
* Interview Insights

### Auto Save

* Automatically saves notes
* Shows save status
* Stores data locally

### Search

Search across:

* Topics
* Patterns
* Notes

### Favorites

Mark important patterns for quick access.

### Question References

Store useful problem links for each pattern.

### Revision Tracking

Track:

* Revision Count
* Last Revised Date

### Dark Mode

Switch between Light and Dark themes.

### Responsive Design

Works on:

* Desktop
* Tablet
* Mobile
